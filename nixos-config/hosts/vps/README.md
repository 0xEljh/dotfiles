## VPS Bootstrap Profile

This profile is the low-risk first boot target for an in-place Ubuntu -> NixOS migration.

### Flake target

- `.#vps-bootstrap`

### Required host files

Before building on the target machine, copy the files generated by `nixos-infect`:

```bash
cp /etc/nixos/hardware-configuration.nix ~/nixos-config/hosts/vps/hardware-configuration.nix
test -f /etc/nixos/networking.nix && cp /etc/nixos/networking.nix ~/nixos-config/hosts/vps/networking.nix
```

`networking.nix` is optional, but recommended on providers that need static networking.

### Set next boot generation

```bash
source /root/.nix-profile/etc/profile.d/nix.sh
NIX_CONFIG="experimental-features = nix-command flakes" \
  /nix/var/nix/profiles/system/sw/bin/nixos-rebuild boot --flake path:/root/nixos-config#vps-bootstrap
```
