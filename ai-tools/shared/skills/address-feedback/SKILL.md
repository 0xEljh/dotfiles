# Address Feedback Skill

Here is feedback from reviewers on a design document.

For each point raised, you may accept or push back, with justification.

Reviewers are not always right. Escalate all non-trivial decisions to me. We can discuss. Use the question asking tool where applicable.

When updating the design document with feedback, do not introduce versioning (that has already been covered by other tools). The document should be self-contained.
